# Rebrickable data attribution

The Rebrickable bulk CSV files mirrored in `rebrickable-*` releases are
sourced from <https://rebrickable.com/downloads/>.

Rebrickable's downloadable database is provided under the Creative Commons
Attribution 2.0 license (CC BY 2.0). Data is © Rebrickable and is
redistributed here under that license, with attribution to Rebrickable.

The committed cross-reference files (`data/rebrickable/part_crossrefs.ron`,
the color reference) contain BrickLink/BrickOwl/LEGO identifier strings.
These are **second-hand via the Rebrickable API** — as Rebrickable reports
them, not authoritative — and no BrickLink/BrickOwl/LEGO service is queried
or scraped by this repo (relevant to the per-source terms-of-service gates,
issue #6).
